# Подбор типографики

Референс использует мягкий геометричный sans с тяжёлыми заголовками и очень читаемыми ценами. Для русского приложения выбран **Manrope**: он поддерживает кириллицу и уже присутствует в дизайн-системе проекта.

- Display / H1: Manrope 800, −0.05em.
- Section title: Manrope 800, −0.03em.
- Product title / CTA: Manrope 700.
- Supporting copy: Manrope 500/600.
- Price: Manrope 800, tabular figures при наличии.

Вторая семья не требуется: один шрифт лучше сохраняет плотность мобильного UI и исключает проблемы с кириллицей.
