# Бриф для собственных продуктовых фото

Фотографировать или генерировать собственные напитки для «Кофе Мама»: вертикальный кадр 4:5, напиток по центру, натуральные материалы, мягкий боковой дневной свет, спокойный фон cream/stone/coffee, реалистичная текстура льда, молочной пены и стекла. Без логотипов сторонних кофеен, без текста на изображении, без рук и телефонов в каталожных карточках. Нужны две версии: светлая нейтральная и тёмная драматичная. Результат хранится в Object Storage в WebP, с alt-текстом на русском.
